Evo KokoroTTS

API REST local de Text-to-Speech multilíngue usando Kokoro-82M e Edge TTS. Converta texto em áudio com múltiplas vozes, controle de velocidade, ajuste de pitch e streaming em tempo real.

ⓘ
Base URL: http://localhost:8880
A API também gera documentação interativa automática em /docs (Swagger UI) e /redoc (ReDoc).

Normalização Unicode (NFC)

ⓘ
A API normaliza automaticamente todos os textos para NFC (forma composta Unicode). Isso garante que caracteres como ã, ç, é, ô, ü sejam tratados corretamente, independentemente de como foram digitados ou copiados. Aspas tipográficas, travessões e reticências Unicode também são convertidos.

Sintetiza o texto completo e retorna o arquivo de áudio.

pitchintnão0Ajuste de pitch em Hz. Faixa: -80 a 80. Em vozes Edge, 0 aplica o preset default.
formatstringnão"mp3""mp3" ou "wav".

  "voice": "pt_f_menina",

  "pitch": 46,

}

Como usar outros idiomas

O idioma da sintese e definido pela voice. Nao existe um campo separado como language no POST de /tts ou /tts/stream.

PassoO que fazer
1Consulte GET /languages para ver os idiomas suportados na instalacao atual.
2Consulte GET /voices e escolha uma voz cujo campo lang seja o idioma desejado.
3Envie o texto usando essa voice. A engine correta (Kokoro ou Edge) sera escolhida automaticamente.
ⓘ
Idiomas configurados nesta versao: pt-br, en-us, en-gb, es-es, fr-fr, ja-jp, zh-cn, hi-in, it-it e de-de.
Exemplos de vozes: pf_dora (pt-BR), af_bella (en-US), bf_alice (en-GB), ef_dora (es-ES), ff_siwis (fr-FR), jf_alpha (ja-JP), zf_xiaoxiao (zh-CN), hf_alpha (hi-IN), if_sara (it-IT) e de_f_katja (de-DE). 
# Ingles (Kokoro)
curl -X POST http://localhost:8880/tts \
  -H "Content-Type: application/json" \
  -d '{"text":"Hello. This is a test in English.","voice":"af_bella","speed":1.0,"format":"mp3"}' \
  --output english.mp3

# Espanhol (Kokoro)
curl -X POST http://localhost:8880/tts \
  -H "Content-Type: application/json" \
  -d '{"text":"Hola. Esta es una prueba en espanol.","voice":"ef_dora","speed":1.0,"format":"mp3"}' \
  --output espanol.mp3

# Alemao (Edge TTS)
curl -X POST http://localhost:8880/tts \
  -H "Content-Type: application/json" \
  -d '{"text":"Hallo. Dies ist ein Test auf Deutsch.","voice":"de_f_katja","speed":1.0,"pitch":0,"format":"mp3"}' \
  --output deutsch.mp3

JavaScript / Fetch

async function speak(text, voice = "pt_f_menina", speed = 1.08, pitch = 46) {
  const response = await fetch("http://localhost:8880/tts", {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ text, voice, speed, pitch, format: "mp3" }),
  });

  const blob = await response.blob();
  new Audio(URL.createObjectURL(blob)).play();
}

Python / Requests

import requests

BASE = "http://localhost:8880"

resp = requests.post(f"{BASE}/tts", json={
    "text": "Olá! Testando a API com voz infantil.",
    "voice": "pt_f_menina",
    "speed": 1.08,
    "pitch": 46,
    "format": "mp3",
})

with open("audio.mp3", "wb") as f:
    f.write(resp.content)

]);
